# Netflix Data Cleaning Project

## Overview

This project focuses on cleaning and preprocessing the Netflix Movies and TV Shows dataset using Python and Pandas in Google Colab.

The goal of this project is to transform raw and inconsistent data into a clean and analysis-ready dataset.

---

## Objectives

* Handle missing values
* Remove duplicate records
* Standardize column names
* Convert date formats
* Fix inconsistent text formatting
* Export cleaned dataset

---

## Technologies Used

* Python
* Pandas
* Google Colab
* Matplotlib
* GitHub

---

## Dataset

Netflix Movies and TV Shows Dataset from Kaggle.

---

## Data Cleaning Steps Performed

### 1. Missing Value Handling

Filled missing values in:

* director
* cast
* country

### 2. Duplicate Removal

Removed duplicate rows using Pandas.

### 3. Column Standardization

Converted column names into lowercase format and removed spaces.

### 4. Date Formatting

Converted date columns into proper datetime format.

### 5. Text Cleaning

Removed extra spaces and standardized categorical values.

---

## Visualization

Created a bar chart showing distribution of Movies and TV Shows available on Netflix.

* Structured dataset for future visualization and machine learning tasks
